## Development

#### Commands

| Command               | Description                           | 
| ----------------------|-------------------------------------- |
| `npm run start`       | Run the current build at `/dist`      |
| `npm run start:quick` | Quick build and start                 |
| `npm run build`       | Build the source code                 |
| `npm run format`      | Prettify source code                  |
| `npm run docs`        | Generate docs                         |
